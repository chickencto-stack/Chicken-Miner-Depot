## Summary

This PR publishes the Windows Chicken Miner v1.4.3 standalone installer ZIP and the full Windows package containing installer, watcher bots, thermal tooling, recommended power settings, documentation, manifests, checksums, and agent prompts.

## Included

- Standalone Windows installer ZIP
- Full Windows installer plus bots, thermal, and power settings ZIP
- Windows watcher/status bot scripts
- Windows thermal status add-on
- Recommended Windows power settings script
- README and installation instructions
- Release notes
- Security policy
- Config preservation notes
- Manifest and SHA256 checksums
- Install-agent prompt
- Commit-agent prompt

## Excluded

- Private fleet tooling
- Private deployment tooling
- Host allowlists
- Private evidence
- Machine-specific configs
- Secrets, tokens, credentials, private keys, wallet config, or pool config
- Defender exclusion automation
- UAC-disable logic
- Out-of-scope automation payloads
- macOS payloads
